# FITMAX GYM WEAR – Premium Gym Wear Landing Page + Admin CMS

Landing page cao cấp cho thương hiệu FITMAX GYM WEAR với trang quản trị CMS đầy đủ tính năng.

## 🚀 Live Demo
[https://fitmax-gym-wear.vercel.app](https://fitmax-gym-wear.vercel.app)

**Admin Panel**: [https://fitmax-gym-wear.vercel.app/admin](https://fitmax-gym-wear.vercel.app/admin)

## ✨ Tính Năng

### Landing Page
- Hero section với ảnh split-layout và floating cards
- Infinite ticker banner
- Brand values cards
- Categories bento grid
- Best sellers với size selector, wishlist, add-to-cart
- Benefits với fabric technology info
- Social proof / reviews
- Flash Sale countdown timer
- Newsletter subscription
- Responsive footer

### Admin CMS
- Đăng nhập bảo mật (JWT cookie)
- Chỉnh sửa tất cả text content
- Upload ảnh trực tiếp
- 10 sections có thể chỉnh sửa

## 🛠 Tech Stack
- Next.js 14 (App Router)
- TypeScript + Tailwind CSS
- JWT Authentication
- Vercel Blob Storage

## 📦 Setup

```bash
npm install
npm run dev
```
